# Infosys FakeJob Detection using NLP

## Overview
This project uses Natural Language Processing (NLP) techniques to detect fake job postings.

## Project Structure
```
Infosys_FakeJob Detection using NLP/
├── frontend/          # Frontend application files
├── model/             # ML model and datasets
│   ├── Fake Postings.csv
│   └── fake_job_postings_800fake.csv
└── README.md
```

## Datasets
- **Fake Postings.csv**: Sample fake job postings dataset
- **fake_job_postings_800fake.csv**: Dataset with 800 fake job postings for training and testing
